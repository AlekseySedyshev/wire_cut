# wire_cut
Wire cutter project, 
CPU - STM32F030, LCD -  LM15SGFN707 (Hynix HM17CM4101)
Step motor - Nema17 (1.8 deg./step) - using with 3d printer extruder driver.
Servo - 25kg (Hobby King HK15338)
Knife - Steel 1066
